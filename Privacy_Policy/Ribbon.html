Last updated: 2026-02-24

Al-Aseel Maqam respects user privacy and is committed to protecting the data collected within the application. This Privacy Policy explains what data is collected, how it is used, and how it is protected.

ENGLISH

Information We Collect

The application collects only the following data:

  • Device Identifier (Phone/Device ID) — Used for app activation and license verification to prevent unauthorized use.
  • Email Address — Used solely for activation management, recovering activation codes if lost, and verifying license ownership when providing support.
  • Phone Number — Provided voluntarily by user for support or activation help.

The application does not collect:

  • Names
  • Location data
  • Photos or files
  • Contacts

Purpose of Data Use

Collected data is used exclusively for:

  • App activation and license management
  • Preventing misuse or activation fraud
  • Providing activation-related technical support

The data is not used for advertising, marketing, or analytics purposes.

Data Sharing

No user data is sold, shared, or rented to any third party.

Data Security

Appropriate technical measures are implemented to protect collected data against unauthorized access or misuse.

App Permissions

The application requests only the permissions necessary for its core functionality, such as MIDI device communication.

Children’s Privacy

Al-Aseel Maqam does not target or knowingly collect personal data from children under the age of 13.

Changes to This Policy

This Privacy Policy may be updated when required. Any changes will be published on this page.

Consent

By using Al-Aseel Maqam, you agree to this Privacy Policy.
