Al-Aseel Maqam

Quarter-tone (¼-tone) tuning + MIDI routing (Badel Router) + Transpose — built for live performance.

Features

  • Professional quarter-tone (¼-tone) tuning for maqam performance
  • Badel Router (MIDI routing / switching) between connected devices
  • Transpose control with quick + / − and instant reset
  • Multi-keyboard support (control more than one keyboard simultaneously)
  • Five Save buttons (1–5) to store and recall presets instantly

Hardware Requirements

  • USB Hub or OTG Hub
  • Two USB printer cables (USB-A to USB-B) — one cable per keyboard (for two keyboards at the same time)

العربية

الأصيل مقام تطبيق أندرويد احترافي للتحكم بالربع تون، وتوجيه إشارات MIDI (راوتر البدل)، وإدارة الترانسبوز أثناء العزف المباشر.

المميزات

  • دعم احترافي للربع تون (¼-Tone) للعزف بالمقامات الشرقية
  • راوتر البدل لتوجيه وتبديل إشارات MIDI بين الأجهزة
  • خاصية الترانسبوز مع الرجوع الفوري إلى الصفر
  • دعم تشغيل أكثر من جهاز أورغ في الوقت نفسه
  • خمسة أزرار للحفظ (1–5) لاسترجاع الإعدادات بسرعة

المتطلبات

  • هاب USB أو OTG Hub
  • عدد 2 كيبل طابعة (USB-A إلى USB-B) لتفعيل جهازين في الوقت نفسه
